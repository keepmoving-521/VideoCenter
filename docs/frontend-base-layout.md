# W02：前端基础页面布局

前端已从初始化欢迎页升级为完整应用壳，为后续影视库、下载管理、播放和任务页面提供统一
布局。

## 布局结构

- 桌面端固定左侧导航；
- 顶部工具栏；
- 独立可滚动内容区；
- 移动端侧栏抽屉和背景遮罩；
- 后端连接状态；
- 搜索入口、通知入口和用户区域；
- 系统设置入口。

## 主导航

当前预留以下页面：

- 首页；
- 影视库；
- 下载管理；
- 本地资源；
- 任务中心；
- 观看历史。

W02 使用组件状态切换占位页面，不额外引入路由依赖。后续正式实现页面时可接入前端路由，
布局结构无需重写。

## 首页内容

首页包含：

- 欢迎与快捷操作区域；
- 系统概览卡片；
- 继续观看区域；
- 实时任务动态区域。

当前统计和列表内容使用占位状态，后续迭代再接入真实 API。

## 响应式设计

- `820px` 以下切换为抽屉导航；
- `580px` 以下压缩顶部工具栏和首页卡片；
- 支持键盘焦点样式；
- 支持系统“减少动态效果”设置；
- 最低支持 `320px` 页面宽度。
